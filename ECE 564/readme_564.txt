(1)weight 3x3
first row: dimension of matrix
second row(7 0s added):0,0,0,0,0,0,0,w22,w21,w20,w12,w11,w10,w02,w01,w00
(2)input(zero padded each line, example 10x10(6 0s added each row for values))
first row(dimen one):	0000_0000_0000_1010
second row(dimen)one:	0000_0000_0000_1010 
thrid row(values):	0,0,0,0,0,0,i09,i08,i07,i06,i05,i04,i03,i02,i01,i00 
...
...
...
(3)output(zero padded each line, example(8x8, 8 0s added each row))
first line:		0,0,0,0,0,0,0,0,o07,o06,o05,o04,o03,o02,o01,o00		 
...
...
...

3. stop reading input
If the data is FF(0000_0000_1111_1111), it means there is no more data, you can stop reading. 
FF should appear in the last row of input data(2c)
